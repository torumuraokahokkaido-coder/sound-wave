🎓 物理法則：音波・デシベル・音高

① 振幅と音の大きさ [デシベル: dB]

「振幅（Amplitude）」は、空気の揺れの「変位の最大値」を表すリニアな量です。 しかし、人間の耳は音の強さを対数スケールで感じ取るため、物理的には**「デシベル（dB）」**という単位を使います。 本アプリでは、入力された波形データ（時間領域）から実効値（RMS）を自前計算し、次の対数公式を用いてデシベルに換算しています：

dB = 20 × log₁₀(RMS)

※無音状態を -60 dB、最大の入力を 0 dB として測定しています。

② 振動数（周波数）[Hz] と音名

音が1秒間に振動する回数を「振動数（周波数）[Hz: ヘルツ]」と呼びます。 今回のゲームでは、日本の童話**『かえるの合唱』**を構成する美しい音階を正確な物理振動数にマッピングしています。

ド (C4) : 約 261.6 Hz
レ (D4) : 約 293.7 Hz
ミ (E4) : 約 329.6 Hz
ファ (F4) : 約 349.2 Hz
ソ (G4) : 約 392.0 Hz
ラ (A4) : 約 440.0 Hz

③ ドラッグによるスケール変形

「時間軸（X軸）スケール」を変えると、波の進行方向への圧縮・伸長（波長の観察）ができます。 「振幅（Y軸）スケール」を変えると、波の縦の振幅をアンプ（増幅器）のように調整して拡大観察できます。 公式 $v = f\lambda$ により、周波数が高い（高音）ほど波長 $\lambda$ が短くなり、画面上でも波がギュッと詰まります。
